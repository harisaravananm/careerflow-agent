# CareerFlow

AI agent for intelligent hiring. Tailors candidate resumes & scores fit.

## Features
- Candidate Agent: Resume tailoring, cover letter generation
- Recruiter Agent: Candidate scoring, skill matching
- Built with Gemini 2.0 Flash

## Setup
```bash
pip install google-generativeai
export GEMINI_API_KEY="your_key"
python careerflow_agent.py
```
